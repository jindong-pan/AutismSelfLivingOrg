# AI工具部署指南 - AI Tools Deployment Guide

## 🎉 已完成的AI工具

您的非营利组织现在拥有两个强大的AI工具来帮助自闭症儿童和家长！

### 1. 📊 日常行为追踪器 (Behavior Tracker)
**位置**: `behavior-tracker/`
**功能**: AI驱动的行为记录、模式分析和个性化建议
**用户**: 自闭症儿童家长

### 2. 🤖 情感支持聊天机器人 (Emotional Support Bot)
**位置**: `emotional-support-bot/`
**功能**: 24/7情感支持、资源推荐和危机检测
**用户**: 自闭症儿童家长

## 🚀 立即开始使用

### 步骤1：测试工具
```bash
# 测试行为追踪器
cd ai-tools/behavior-tracker
python demo.py

# 测试情感支持机器人
cd ../emotional-support-bot
python demo.py
```

### 步骤2：安装依赖（可选）
```bash
# 为行为追踪器安装额外依赖
cd ai-tools/behavior-tracker
pip install -r requirements.txt
```

### 步骤3：开始使用
```bash
# 启动行为追踪器
cd ai-tools/behavior-tracker
python behavior_tracker.py

# 启动情感支持机器人
cd ../emotional-support-bot
python emotional_support_bot.py
```

## 🔧 自定义和扩展

### 添加新功能
```python
# 在行为追踪器中添加新行为类别
tracker.behavior_categories["new_category"] = ["关键词1", "关键词2"]

# 在情感支持机器人中添加新资源
bot.resources["new_category"] = ["资源1", "资源2", "资源3"]
```

### 本地化
- 添加更多中文关键词
- 包含本地资源和支持团体
- 适应当地文化习俗

### 数据隐私
- 所有数据存储在本地
- 支持数据导出和删除
- 符合隐私保护法规
