# 自闭症儿童家长情感支持聊天机器人

## 🎯 概述

这是一个专门为自闭症儿童家长设计的情感支持聊天机器人，提供24/7的情感支持、资源推荐和实用建议。通过AI驱动的情感识别，为疲惫的家长提供及时的支持和鼓励。

## ✨ 核心功能

### 🤖 智能情感识别
- 自动识别用户的情绪状态（疲惫、焦虑、孤独、沮丧、内疚等）
- 基于情绪提供个性化支持响应
- 跟踪对话历史以提供连续支持

### 💬 个性化支持响应
- **疲惫**: 鼓励休息和自我照顾
- **焦虑**: 提供缓解技巧和专业帮助建议
- **孤独**: 推荐家长支持团体
- **沮丧**: 帮助重塑积极视角
- **内疚**: 提供自我宽恕和现实期望

### 🔗 资源连接
- **支持团体**: 家长互助群、本地社区
- **专业帮助**: 行为分析师、心理医生、治疗师
- **临时照顾**: 喘息服务、周末营地
- **教育资源**: 书籍、在线课程、研讨会

### 🚨 危机检测
- 自动识别危机关键词
- 提供紧急联系方式
- 引导寻求专业帮助

## 🚀 快速开始

### 运行演示
```bash
python demo.py
```

### 启动聊天机器人
```bash
python emotional_support_bot.py
```

### 交互命令
- 输入您的感受或问题
- 输入 `summary` 查看对话总结
- 输入 `quit` 或 `退出` 结束对话

## 💡 使用示例

### 典型对话流程：
```
您: 我今天感到很累，照顾孩子太辛苦了
机器人: 我理解照顾自闭症儿童会让人感到非常疲惫。您正在做一件了不起的事情！💪

您: 我觉得很孤独，没有人理解我的感受
机器人: 加入家长互助群，可以和其他经历相似的家长交流经验。

您: 我需要一些资源来帮助孩子
机器人: 临时照顾资源：
• 家庭支持服务
• 临时照顾服务
```

## 🔒 隐私与安全

- **本地处理**: 所有对话都在本地处理，不上传到服务器
- **匿名支持**: 不收集个人信息
- **危机响应**: 在检测到危机情况时提供专业帮助指引
- **尊重界限**: 机器人不是专业医疗建议的替代品

## 🎯 设计理念
