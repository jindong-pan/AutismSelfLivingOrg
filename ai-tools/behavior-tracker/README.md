# 自闭症儿童日常行为追踪器

## 🎯 概述

这是一个AI驱动的行为追踪工具，专门为自闭症儿童的家长设计。通过记录和分析日常行为模式，提供个性化的洞察和建议，帮助家长更好地理解和支持孩子。

## ✨ 主要功能

- **📝 智能行为记录**: 记录行为、强度、触发因素和持续时间
- **📊 实时分析**: 自动分析行为模式和趋势
- **💡 个性化建议**: 基于数据提供针对性建议
- **📈 进展跟踪**: 可视化儿童的发展轨迹
- **📋 自动报告**: 生成专业的行为分析报告

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python behavior_tracker.py
```

### 3. 开始记录

程序启动后，按照提示进行操作：
- 选择"1"添加新行为记录
- 选择"2"查看今日总结
- 选择"3"分析行为模式
- 选择"4"生成报告

## 📖 使用指南

### 行为记录

记录时需要提供：
- **行为描述**: 具体发生了什么（例如："大声尖叫"、"专注玩拼图"）
- **类别**: positive（积极）、challenging（挑战性）、developmental（发展性）
- **强度**: 1-5级（1最轻微，5最严重）
- **触发因素**: 可能的诱因（可选）
- **备注**: 额外观察或上下文
- **持续时间**: 行为持续的分钟数

### 分析功能

程序会自动分析：
- **时间模式**: 找出最活跃的时间段
- **触发因素**: 识别常见诱因
- **强度趋势**: 分析各类行为的严重程度
- **个性化建议**: 基于模式提供具体建议

## 📊 示例场景

### 记录示例：
```
行为描述: 在超市排队时发脾气
类别: challenging
强度: 4
触发因素: 等待时间长
备注: 孩子不喜欢排队，提前准备了玩具但还是发生了
持续时间: 15
```

### AI分析结果：
```
最活跃时间: [14, 16, 18] (下午2-6点)
常见触发因素:
  • 环境变化: 8次
  • 社交压力: 5次
  • 日常变化: 3次

建议:
1. 针对'环境变化': 尝试在日常变化前提前准备和过渡活动
2. 针对'社交压力': 逐渐增加社交活动，从小群体开始
3. 考虑咨询行为分析师，制定行为干预计划
```
