# 🌐 AutismSelfLivingOrg 官方网站

## 🎯 网站目标

创建专业的非营利组织网站，展示AI赋能的自闭症支持服务，吸引捐赠者、合作伙伴和需要帮助的家庭。我们独特的理念是：**不试图"正常化"自闭症儿童，而是支持他们独立生活、理解他们、帮助他们适应生活，让他们生活得更幸福**。

## 🛠️ 技术栈选择

### 推荐方案：Next.js + Tailwind CSS

**优势**:
- ⚡ 快速加载和SEO友好
- 📱 完全响应式设计
- 🎨 现代化UI组件
- 🔧 易于维护和扩展
- 💰 成本效益高

**替代方案**:
- WordPress (更简单，但定制性较低)
- React Static Site (适合纯展示网站)

## 📋 网站结构

### 核心页面
1. **首页** - 使命陈述、AI工具展示、号召行动
2. **关于我们** - 组织故事、团队、使命愿景
3. **AI工具** - 行为追踪器、情感支持机器人介绍
4. **服务** - 主动支持项目、合作伙伴网络
5. **筹款** - 捐赠页面、资助机会
6. **资源** - 家长指南、自闭症资源链接
7. **博客** - 项目更新、成功故事、教育内容
8. **联系我们** - 联系表单、加入我们

### 特殊功能
- 🏦 **捐赠按钮** - 集成支付系统
- 📧 **通讯订阅** - Mailchimp集成
- 📊 **影响数据** - 实时展示项目影响
- 🌍 **多语言支持** - 中英文切换
- 📱 **移动优化** - 响应式设计
